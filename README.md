# Paramjeet Portfolio

This repository contains the source code for my personal portfolio website, built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- About, skills, projects, certifications, and contact sections
- Resume download link
- Social links to GitHub and LinkedIn

## Run locally
Open `index.html` in your browser, or serve the folder with any static file server.

## Deploy
This site can be published easily with GitHub Pages.
